Stock sounds for `POST /api/audio/play`. The bundled `.wav` files ship with the BUSY Bar firmware (© Flipper FZCO, CC-BY-SA 4.0 — see `docs/ATTRIBUTION.md`).

`stock_path` accepts the hardware form (`"shared/volume_change.wav"` — the firmware uses the basename after the last `/`, extension included) as well as the bare name without extension (`"volume_change"`, emulator-only convenience).

Drop extra `.wav`/`.mp3`/`.ogg` files here and restart the server to add your own.
